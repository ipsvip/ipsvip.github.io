sublime text3刚安装完成之后打开文件夹显示的样式如下

这样子看起来眼睛会很难受,那么我们怎样才能使得引导菜单的颜色和编辑器底色一样呢?

解决方法如下:

Ctrl+Shift+P -> install -> 搜索安装包SyncedSidebarBg，自动同步侧边栏底色为编辑窗口底色。
PS：有时改完后侧边栏颜色没变化，不知什么原因，打开包控制，然后列一下已安装包就刷新了。
